# EmailToOneDrive-AzureLogicApp
Automatically save incoming Outlook emails to OneDrive as text files using Azure Logic Apps without writing code.
Technology
- Microsoft Azure Logic Apps
- Office 365 Outlook Connector
- Microsoft OneDrive
- Azure Resource Group
  workflow
1. New email arrives in Outlook.
2. Azure Logic App is triggered.
3. Email Subject and Body are extracted.
4. Text file is created in OneDrive.
5. Email content is backed up automatically.
